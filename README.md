# frc back-end API

income API Refer

* get  /income 查询
* post /income 创建
* put  /income 更新
* post /delete 删除

* get /incomeTotal


## 数据库: mongodb

使用 [mongoose](http://mongoosejs.net/)

> elegant mongodb object modeling for node.js

Mongoose为模型提供了一种直接的，基于scheme结构去定义你的数据模型。它内置数据验证， 查询构建，业务逻辑钩子等，开箱即用。
